About This Project

This interactive website showcases various animals and their colors. Users can explore different animals by clicking color buttons or using keyboard shortcuts. Each animal comes with detailed information about their habitat, behavior, and interesting facts.

How to Use

  • Click on the color buttons in the header or use keyboard shortcuts
  • Press the corresponding letter key for each color (e.g., 'r' for red)
  • Click "View Details" to learn more about each animal
  • Navigate back using the back button in the top-left corner

Technologies Used

  • HTML5
  • Tailwind CSS
  • JavaScript
